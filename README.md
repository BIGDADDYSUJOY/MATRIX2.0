## 🚀 Run and Deploy Locally

This contains everything you need to run the **Atomic Lay Line Decoder** locally.

### Prerequisites
*   **Node.js**
*   **pnpm** (preferred) or npm

### Setup Instructions
1.  **Install dependencies:**
    ```bash
    pnpm install
    ```
    *(or `npm install`)*
2.  **Configure API Key:**
    Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key.
3.  **Run the app:**
    ```bash
    pnpm run dev
    ```

### ⌨️ Keyboard Controls
*   `[SPACE]`: Toggle Wave Mode (Traveling / Standing)
*   `[F] / [D]`: Increase / Decrease Frequency
*   `[I] / [O]`: Increase / Decrease Intensity
*   `[C] / [V]`: Increase / Decrease Chaos
*   `[R]`: Reset Parameters to Baseline
